3 Nettoyage des données et prétraitement de texte
¶

L’objectif de cette partie est de nettoyer le texte afin de faciliter l’apprentissage. Pour entraînerWord2Vec, il est préférable de ne pas supprimer les mots vides car l’algorithme s’appuie sur lecontexte plus large de la phrase afin de produire des vecteurs de mots de haute qualité.En premier temps vous allez créer une fonction qui prend en paramètres un review et qui renvoiele review nettoyé (liste des mots). Pour ce faire:

1. Tout d’abord vous allez supprimer les balises HTML. Pour cela, vous utiliserez la biblio-thèque Beautiful Soup. Si vous n’avez pas installé Beautiful soupe, faites: “pip install BeautifulSoup4”
2. Pour supprimer la ponctuation et les nombres, vous allez utiliser un module pour traiter lesexpressions régulières, appelé re. Optez par la fonction sub() du module re.
3. Par la suite, vous allez convertir les mots en minuscules et appliquer un split pour les sé-parer.

import string
import re
def convert_txt(p):
    p = p.lower()    
    p = re.sub("[^a-zA-Z']", " ", p)
    p = re.sub("<br\\s*/?>", " ", p)
    for i in string.punctuation:
        p = p.replace(i,'')
    p = re.sub("¨", "", p)
    p = p.split(" ")
    return p
    

print("Done")

  2. Ensuite, définir une fonction qui prend en paramètre le paragraphe du review et le tokenizer et qui renvoie la liste des phrases, tel que chaque phrase est une liste des mots. Pensez à utiliser la fonction précédente.

def pretraitement(text):
    text = tokenizer.tokenize(text)
    tmp=[]
    for p in text:
        convert_txt(p)
        p = list(filter(None, p))
        tmp.append(p)
    return tmp

4 Apprentissage et enregistrement de votre modèle¶

Le choix des paramètres n’est pas facile, mais une fois que vous avez choisi vos paramètres, créerun modèle Word2Vec est simple, pour ce faire:

  1. Initialisez et entraînez votre modèle en utilisant l’objet “word2vec” du sous-module “models”du module “gensim”

  2. Enregistrez ce modèle dans un fichier après l’avoir entraîner.

import multiprocessing

cores = multiprocessing.cpu_count()
