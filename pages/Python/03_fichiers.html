Les fichiers¶

Lecture et écriture dans un fichier¶

Les modes d’ouverture¶

  • 'r' ouverture en lecture (Read).

  • 'w' ouverture en écriture (Write). Le contenu du fichier est écrasé. Si le fichier n’existe pas, il est créé.

  • 'a' ouverture en écriture en mode ajout (Append). On écrit à la fin du fichier sans écraser l’ancien contenu du fichier. Si le fichier n’existe pas, il est créé.

On peut ajouter à tous ces modes le signe b pour ouvrir le fichier en mode binaire.

import os
os.chdir("..\..\..\P03_mail")

mon_fichier = open("apprenantmail.txt", "r")
# on a ouvert le fichier en mode lecture

# il ne faut pas oublier de fermer le fichier
mon_fichier.close()

Fonction Write¶

La méthode write n’accepte en paramètre que des chaînes de caractères. Pour écrire des nombres dnas un fichier, il vous faut les convertir en chaîne avant de les écrire et les reconvertir en entier après les avoir lus.

mon_fichier = open("readme.txt", "w") # si le fichier n'existe pas, il est créé dans le dossier courant

mon_fichier.write("Le fichier 'apprenantmail.txt' contient la liste des adresses mail de tous les apprenants")

mon_fichier.close()

# on lit le fichier pour vérifier qu'il a bien été créé et modifié en écriture

mon_fichier = open("readme.txt", "r")
contenu = mon_fichier.read()
print(contenu)
mon_fichier.close()

Manipulation de fichiers avec With¶

Pour éviter les erreurs de lecture et d’écriture des fichiers, il est nécessaire de les fermer dès qu’on en a plus besoin. Ci-dessus, nous avons explicitement demandé la fermeture avec la fonction close(). Cependant, lorsque l’on code des programmes qui nécesitent la manipulation de plusieurs fichiers, on peut rapidement oublier de les refermer.

Pour éviter cette situation, on utilise le plus souvent la fonction with, avec la syntaxe suivante : with open(mon_fichier, mode_ouverture) as variable:

with open("readme.txt", "r") as f:
    contenu = f.read()
    print(contenu)

Enregistrer des objets dans des fichiers¶

On peut enregistrer des objets dnas des fichiers grâce au module pickle, en utilisant deux classes incluses dans ce module : la classe Pickler et la classe Unpickler, et les méthodes dump et load.

Enregistrement : utilisation de Pickler et de dump :

import pickle

donnees = [{"Nantes": "Loire Atlantique"}, {"Lyon" : "Rhône"}, {"Brest" : "Finistère"}]


with open("donnees.txt", "wb") as f:
    mon_pickler = pickle.Pickler(f)
    mon_pickler.dump(donnees)

Récupération des objets enregistrés : utilisation de Unpickler et de load :

with open('donnees.txt', 'rb') as f: 
    mon_depickler = pickle.Unpickler(f)
    donnees_recuperees = mon_depickler.load()
    print(donnees_recuperees)
