Consommer une WEB API avec Python¶

Dans ce TP, vous allez utiliser Python pour afficher un graphique sur des températures. Les données à exploiter sont obtenues à partir d’une WEB API.

Requêter la WEB API¶

Voici le code pour requêter la WEB API d’ OpenWeather. La bibliothèque Requestpermet de générer une requête HTTP et d’en récupérer la réponse. La bibliothèque jsonde traiter ce type de fichier, qui est donné dans la réponse de la WEB API.

import json
import requests
import pprint

owm_url = "https://samples.openweathermap.org/data/2.5/forecast?lat=0&lon=0&appid=xxx"
reponse = requests.get(owm_url)
contenu = json.loads(reponse.content.decode("utf-8"))

pprint.pprint(contenu)
