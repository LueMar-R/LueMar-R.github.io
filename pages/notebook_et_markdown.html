Insertion d’images et de liens hypertextes ¶

Voici la syntaxe générale.

[Je suis un lien](https://github.com)  ![Je suis une image de Moufette](https://upload.wikimedia.org/wikipedia/commons/5/57/Striped_Skunk.jpg)

Pour plus de possibilités, voir ce lien vers le cheatset Markdown de GitHub

Je suis un lien

L’image est automatiquement redimensionnée pour s’ajouster à la page du notebook.

Markdown ne permet pas de redimensionner directement l’image (sauf dans certaines implémentations sur macOS). Il est alors nécessaire d’utiliser des balises HTML :
<img src="https://upload.wikimedia.org/wikipedia/commons/5/57/Striped_Skunk.jpg" alt="moufette" width="200"/>
